# الكبائن الفاخرة | Luxury Cabins

موقع إلكتروني احترافي لمؤسسة الكبائن الفاخرة للمقاولات العامة — تأجير وبيع وتصنيع الوحدات المتنقلة في المملكة العربية السعودية.

## التشغيل المحلي

```bash
npm install
npm run dev
```

افتح [http://localhost:3000](http://localhost:3000)

## الصفحات

- `/` الرئيسية
- `/about` من نحن
- `/rental` التأجير
- `/manufacturing` البيع والتصنيع
- `/projects` المشاريع والعملاء
- `/faq` الأسئلة الشائعة
- `/contact` اتصل بنا

## ملاحظات للعميل

- حدّث رقم الجوال والواتساب والبريد في `src/lib/site.ts`
- استبدل صور المشاريع وشعارات العملاء بالمحتوى الفعلي
- محتوى صفحة البيع والتصنيع قابل للتحديث عند تزويدكم بالنصوص النهائية
